Wildlife Tracker - <b>Work in Progress</b>

There are 2 different tables. One table lists the species. The second table lists the sightings with Date, longitude and latitude. 

Validations include: 
* Species name field cannot be blank and cannot contain numerical value
* Data cannot be in the future.
* Longitude and Latitude cannot be blank and has a numerical value
* Species has one
* Sightings has many
